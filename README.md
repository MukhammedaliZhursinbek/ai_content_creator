  AI Content Generator



AI Content Generator — это веб-приложение, которое использует искусственный интеллект для генерации текста на заданные темы в различных стилях. Приложение позволяет создавать тексты в формате рекламы, формальных статей или разговорных блогов на основе параметров, таких как тема, стиль, длина текста и язык.



   Возможности



\- Генерация текста на основе темы, стиля и языка.

\- Поддержка нескольких стилей генерации текста:

  - Рекламный стиль

  - Формальный стиль

  - Разговорный стиль

\- Ограничение на количество слов в сгенерированном тексте.



   Установка и запуск



1\. Клонировать репозиторий



   ```bash

   git clone https://github.com/MukhammedaliZhursinbek/ai\_content\_creator.git

   cd ai-content-generator



2\. Создание виртуального окружения

   python -m venv venv



   Для активации в Windows:
venv\\Scripts\\activate



   Для активации в Linux/Mac:

   source venv/bin/activate





3\. Установка зависимостей

   pip install -r requirements.txt





4\. Настройка переменных окружения
Необходимо получить API-ключ с OpenAI. После создать файл .env в корне проекта
OPENAI\_API\_KEY=your-api-key-here





5\. Запуск проекта

   В виртуальном окружении venv нужно запустить с помощью команды

   python main.py
